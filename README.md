# Lab8-Starter

By Antonio Didjusto

Graceful degradation is the practice of building an application so that it still functions at a basic level, even when advanced features are unavailable. Service workers support graceful degradation because they allow the web app to continue working offline, or when network connections are slower. Instead of failing completely with no internet, it serves cached files and data, providing an experience that is still usable even if it is not as good.
